Задание 2: Работа с DOM

Цель задания

Освоить работу с Document Object Model (DOM) для динамического изменения содержимого веб-страницы с использованием JavaScript.

Задачи

  1. Создать два массива: латинских высказываний и их переводов
  2. Реализовать вывод фраз в случайном неповторяющемся порядке
  3. Применить чередование классов при выводе фраз
  4. Добавить функционал изменения стиля четных строк
  5. Создать динамические элементы с форматированием текста

Требования

  • Использовать методы работы с DOM (createElement, appendChild, etc.)
  • Применить случайную выборку без повторений
  • Реализовать чередование классов class1 и class2
  • Использовать CSS для стилизации четных элементов
  • Создать структуру с форматированием текста программно

Задание 1: Flexbox Анимация

Цель задания

Освоить работу с CSS Flexbox и создание интерактивных анимаций с использованием CSS transitions и keyframes. Задание демонстрирует возможности современного CSS для построения компактных блоков с реакцией на действия пользователя.

Задачи

  1. Создать flexbox-контейнер размером не более 200px.
  2. Разместить три изображения: статичный набор посуды, кипящую кастрюлю и крышку.
  3. При наведении мыши — медленно проявлять кастрюлю с увеличением прозрачности и размера.
  4. При нажатии — анимация хлопающей крышки, пока удерживается нажатие.
  5. Использовать простые CSS-свойства без тяжёлых библиотек.

Требования

  • Использовать display: flex для контейнера и выравнивания элементов.
  • Применить transition и @keyframes для плавных эффектов.
  • Позиционирование наложенных картинок через position: absolute.
  • Настройка transform-origin для корректного масштабирования кастрюли.

Интерактивность

При наведении курсора кастрюля плавно появляется и слегка «дышит» (увеличение и уменьшение масштаба). При удержании клика крышка хлопает — поднимается и опускается с поворотом. В момент нажатия кастрюля остаётся видимой в исходном положении. Размер блока с демонстрацией не превышает 200px по условию задания.

Критерии оценки

  • Корректное использование Flexbox для размещения и центрирования.
  • Плавность анимаций и отсутствие рывков при наведении и клике.
  • Правильное позиционирование всех трёх изображений в одном блоке.
  • Работоспособность эффектов наведения и удержания кнопки мыши.
  • Чистота кода и использование только CSS без лишних зависимостей.
